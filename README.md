# Élan B2

Personal DELF B2 study coach built with Next.js and TypeScript.

## Features

- Mobile-first practice for listening, reading, writing, speaking, and grammar
- Non-repeating DELF-style task generation
- Local coach feedback for productive tasks
- Mock exam mode, review mode, and progress tracking

## Run locally

```powershell
& 'C:\Program Files\nodejs\npm.cmd' install
& 'C:\Program Files\nodejs\npm.cmd' run dev
```

Then open `http://localhost:3000`.

## Notes

- This app uses original/generated practice content.
- Progress is currently stored in the browser with `localStorage`.
- External resources such as RFI and TV5MONDE are linked as references only.
